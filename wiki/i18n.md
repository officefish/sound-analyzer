## Введение

Интернационализация (i18n) — процесс адаптации приложения для использования на разных языках.

В SoundLab мы поддерживаем:
- 🇷🇺 Русский (ru) — основной язык разработки
- 🇬🇧 Английский (en) — вторичный язык

Архитектура позволяет легко добавлять новые языки.

---

## Выбор технологии

Для интернационализации выбран **i18next** — наиболее популярное решение в React-экосистеме.

### Почему i18next?

| Критерий | i18next | react-intl | react-i18n |
|----------|---------|------------|------------|
| Популярность | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐ |
| Поддержка React | ✅ | ✅ | ✅ |
| TypeScript | ✅ | ✅ | ⚠️ |
| Плагины | ✅ | ❌ | ❌ |
| Асинхронная загрузка | ✅ | ❌ | ❌ |
| Интерполяция | ✅ | ✅ | ✅ |
| Плюрализация | ✅ | ⚠️ | ⚠️ |

**Итог:** i18next — наиболее гибкое и мощное решение.

## Установка и настройка

### 1. Установка зависимостей

```bash
npm install i18next react-i18next i18next-browser-languagedetector
# или
yarn add i18next react-i18next i18next-browser-languagedetector
```

### 2. Создание конфигурации

```typescript
// src/i18n/config.ts
import i18n from 'i18next';
import { initReactI18next } from 'react-i18next';
import LanguageDetector from 'i18next-browser-languagedetector';

// Импорт переводов
import ruTranslation from './locales/ru/translation.json';
import enTranslation from './locales/en/translation.json';

// Ресурсы переводов
const resources = {
  ru: {
    translation: ruTranslation,
  },
  en: {
    translation: enTranslation,
  },
};


i18n
  // Автоматическое определение языка браузера
  .use(LanguageDetector)
  // Подключение к React
  .use(initReactI18next)
  // Инициализация
  .init({
    resources,
    fallbackLng: 'ru', // Язык по умолчанию
    debug: process.env.NODE_ENV === 'development',
    
    interpolation: {
      escapeValue: false, // React уже экранирует
    },
    
    detection: {
      order: ['localStorage', 'navigator'],
      caches: ['localStorage'],
    },
  });

export default i18n;
```

3. Подключение в приложении

```tsx
// src/main.tsx
import React from 'react';
import ReactDOM from 'react-dom/client';
```

## Динамическая смена языка

### Хук для смены языка

```typescript
// src/hooks/useLanguage.ts
import { useTranslation } from 'react-i18next';
import { useCallback } from 'react';

export const useLanguage = () => {
  const { i18n } = useTranslation();
  
  const currentLanguage = i18n.language;
  const supportedLanguages = ['ru', 'en'];
  
  const changeLanguage = useCallback(async (lang: string) => {
    if (!supportedLanguages.includes(lang)) return;
    await i18n.changeLanguage(lang);
    localStorage.setItem('i18nextLng', lang);
  }, [i18n]);
  
  const toggleLanguage = useCallback(async () => {
    const newLang = currentLanguage === 'ru' ? 'en' : 'ru';
    await changeLanguage(newLang);
  }, [currentLanguage, changeLanguage]);
  
  return {
    currentLanguage,
    supportedLanguages,
    changeLanguage,
    toggleLanguage,
  };
};
```


### 2. Типизация переводов

```typescript
// src/i18n/types.ts
import ruTranslation from './locales/ru/translation.json';

type DeepKeys<T> = T extends object
  ? {
      [K in keyof T]: K extends string
        ? T[K] extends object
          ? `${K}.${DeepKeys<T[K]>}`
          : K
        : never;
    }[keyof T]
  : never;

export type TranslationKey = DeepKeys<typeof ruTranslation>;

// Использование
const { t } = useTranslation();
t('microphone.buttons.start' as TranslationKey);
```

### 3. Хук с типизацией

```typescript
// src/hooks/useTypedTranslation.ts
import { useTranslation } from 'react-i18next';
import { TranslationKey } from '../i18n/types';

export const useTypedTranslation = () => {
  const { t, i18n } = useTranslation();
  
  const typedT = (key: TranslationKey, options?: any) => {
    return t(key, options);
  };
  
  return { t: typedT, i18n };
};
```

### 4. Компонент-обёртка для перевода

```tsx
// src/components/ui/TranslatedText.tsx
import React from 'react';
import { useTranslation } from 'react-i18next';

interface TranslatedTextProps {
  path: string;
  values?: Record<string, any>;
}

const TranslatedText: React.FC<TranslatedTextProps> = ({ path, values }) => {
  const { t } = useTranslation();
  return <>{t(path, values)}</>;
};

export default TranslatedText;
```

## Тестирование интернационализации

### 1. Тестирование переключения языка

```typescript
// src/i18n/__tests__/i18n.test.ts
import i18n from '../config';

describe('i18n', () => {
  beforeEach(() => {
    i18n.changeLanguage('ru');
  });
  
  it('should have Russian translations', () => {
    expect(i18n.t('common.save')).toBe('Сохранить');
    expect(i18n.t('microphone.title')).toBe('Микрофон');
  });
  
  it('should switch to English', async () => {
    await i18n.changeLanguage('en');
    expect(i18n.t('common.save')).toBe('Save');
    expect(i18n.t('microphone.title')).toBe('Microphone');
  });
  
  it('should handle interpolation', () => {
    expect(i18n.t('microphone.volume_level', { percent: 75 }))
      .toBe('Громкость: 75%');
  });
});
```

```tsx
// src/modules/Microphone/__tests__/Microphone.test.tsx
import { render, screen } from '@testing-library/react';
import { I18nextProvider } from 'react-i18next';
import i18n from '../../../i18n/config';
import Microphone from '../index';

describe('Microphone', () => {
  const renderWithI18n = (component: React.ReactNode) => {
    return render(
      <I18nextProvider i18n={i18n}>
        {component}
      </I18nextProvider>
    );

  it('should display Russian text by default', () => {
    renderWithI18n(<Microphone />);
    expect(screen.getByText('Микрофон')).toBeInTheDocument();
    expect(screen.getByText('Включить')).toBeInTheDocument();
    expect(screen.getByText('Выключить')).toBeInTheDocument();
  });
  
  it('should display English text after language change', async () => {
    await i18n.changeLanguage('en');
    renderWithI18n(<Microphone />);
    expect(screen.getByText('Microphone')).toBeInTheDocument();
    expect(screen.getByText('Start')).toBeInTheDocument();
    expect(screen.getByText('Stop')).toBeInTheDocument();
  });
});
```
