## 📋 Обзор

Кроссплатформенное десктопное приложение для обнаружения лая собаки с использованием:
- **Этапы 1-3:** Спектральный анализ (FFT, гармоники, анализ громкости)
- **Этап 4:** Плагинная архитектура + нейросети на TensorFlow.js

**Технологический стек:** TypeScript, Electron/Tauri, Web Audio API, TensorFlow.js
